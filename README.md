# 📌 Zomato Food Delivery Data Analysis & Dashboard

## 📊 Project Overview
This project focuses on analyzing food delivery data inspired by a Zomato-like platform. The objective is to extract meaningful business insights using a complete data analytics workflow, including data cleaning, SQL analysis, Python-based exploration, and interactive Power BI dashboards.

---

## 🎯 Objectives
- Analyze delivery performance across cities  
- Understand customer satisfaction (ratings)  
- Identify cost vs value relationships  
- Evaluate operational efficiency (distance vs delivery time)  
- Build an interactive dashboard for decision-making  

---

## 🛠️ Tools & Technologies Used
- Excel → Data Cleaning & Validation  
- PostgreSQL → Data Querying & Analysis  
- Python (Pandas, Matplotlib, Seaborn) → Data Analysis & Visualization  
- Power BI → Dashboard & Data Visualization

## 📈 Dashboard Insights

### 🔹 Performance Analysis Dashboard

---

## 🔍 Key Insights
- Delivery time increases with distance, showing operational dependency  
- Higher cost does not always guarantee higher customer ratings  
- Certain cities show higher variability in delivery performance  
- Cost segmentation helps identify value-for-money zones  
- Majority of orders fall within mid-range delivery time
